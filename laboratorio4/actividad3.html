Listas ordenadas

Las listas ordenadas en HTML son una herramienta muy útil que nos permite organizar de manera secuencial una serie de elementos, por ejemplo:

comidas rapidas

  1. hamburguesa
  2. pizza
  3. perro calientw
  4. salchipapa
  5. patacon relleno

Listas desordenadas

son utilizadas para crear listas sin ningún orden específico. Los elementos en estas listas aparecerán en el orden en el que se establezcan. Cada elemento de la lista se define mediante la etiqueta, por ejemplo:

Lenguajes de programacion

  • javascript
  • python
  • visual code
  • c++
  • C sharp

Listas anidadas

es una lista dentro de otra lista. Esto significa que podemos tener elementos secundarios (subelementos) dentro de elementos primarios (elementos principales). Esta estructura jerárquica permite organizar la información de manera lógica y visualmente atractiva. Por ejemplo:

construccion de casas

  • recursos
    1. ladrillos
    2. cemento
    3. arenila
  • mano de obra
    1. obreros
      • ladrillos
      • cemento
      • arenilla
      • apilar
    2. arquitecto
    3. conductores
      • camion
      • automovil
      • volquetas
      • hormiguera

Tablas

Una tabla es un conjunto estructurado de datos distribuidos en filas y columnas (datos tabulados). Una tabla permite buscar con rapidez y facilidad valores entre diferentes tipos de datos que indiquen algún tipo de conexión. Por ejemplo:

Nombre Apellido Cedula
Juan Perez 123
Camila Torres 124
Victor Moreno 125
Antonio Moscote 126
